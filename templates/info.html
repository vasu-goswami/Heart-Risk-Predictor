Heart Disease Attributes and Value Ranges

Heart Disease Dataset Attributes
Attribute Explanation Value Range
age Age of the patient (in years). 29 to 77 (years)
sex Sex of the patient. 1 = Male, 0 = Female
cp Chest pain type. 0 = No pain, 1 = Typical angina, 2 = Atypical angina, 3 = Non-anginal pain, 4 = Asymptomatic
trestbps Resting blood pressure (in mm Hg). 94 to 200 (mm Hg)
chol Serum cholesterol (in mg/dl). 126 to 564 (mg/dl)
fbs Fasting blood sugar > 120 mg/dl (1 = True, 0 = False). 1 = True, 0 = False
restecg Resting electrocardiographic results. 0 = Normal, 1 = Having ST-T wave abnormality, 2 = Showing probable or definite left ventricular hypertrophy
thalach Maximum heart rate achieved (in beats per minute). 71 to 202 (bpm)
exang Exercise induced angina (1 = Yes, 0 = No). 1 = Yes, 0 = No
oldpeak Depression induced by exercise relative to rest (ST depression, a marker for heart disease risk). 0.0 to 6.2 (in mm)
slope Slope of the peak exercise ST segment (related to the recovery of the heart after exercise). 1 = Upsloping, 2 = Flat, 3 = Downsloping
ca Number of major vessels colored by fluoroscopy (0-3). 0 to 3
thal Thalassemia (a type of anemia). 3 = Normal, 6 = Fixed defect, 7 = Reversable defect
target Presence or absence of heart disease. 0 = No heart disease, 1 = Heart disease
